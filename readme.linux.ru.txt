CTESK 2.9
============

Требования
----------

1. Core utilities + gcc, make, gdb
2. Sun J2SE 6.0 и выше (http://java.sun.com)
3. Eclipse RCP/Platform Runtime Binary 3.4 и выше (http://eclipse.org)
4. Eclipse C/C++ Development Tools 5.0.1 и выше (http://eclipse.org)
   (Eclipse CDT Testing Feature зависит от org.junit плагина.
   Если плагин org.junit не установлен, компонент
   CDT Testing Feature также не должен устанавливаться.)
5. UniTESK Reports 2.7 и выше (http://forge.ispras.ru/projects/utt)


Установка
---------
1. В каталоге установки Eclipse существует подкаталог dropins.
   Архив CTESK следует распаковать в этот каталог.
   Для облегчения обновления инструмента рекомендуется переименовать
   находящийся внутри архива каталог ctesk-${version} в ctesk
   (или иным способом исключить номер версии из имени каталога).
   Скрипт 'bin/sec.sh' запускает CTESK транслятор из командной строки.
2. Установить переменную окружения CTESK_HOME - путь к распакованному архиву.
3. Добавить каталог %CTESK_HOME%/bin в PATH.
   (Это условие может быть снято в будущих версиях).
4. Установить UniTESK Reports для генерации отчетов о результатах тестирования.


Обновление
----------

1. Перед установкой новой версии инструмента небходимо закрыть Eclipse.
2. Удалить содержимое каталога, в который установлена предыдущая версия CTESK.
3. Запустить и закрыть Eclipse.
4. Перед установкой новой версии инструмента необходимо привести систему
   в соответствие Требованиям, указанным в readme новой версии.
5. Установить новую версию CTESK как описано в разделе Установка.
